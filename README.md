# 🔐 Authentication System (Node.js + JWT)

A production-style authentication system built using Node.js, Express, MongoDB, and JWT. This project demonstrates secure user authentication, token-based authorization, and scalable backend architecture.

---

## 🚀 Features

- User Signup & Login
- Password Hashing using bcrypt
- JWT Authentication (Access Token)
- Refresh Token Flow (Token Regeneration)
- Protected Routes (Middleware-based)
- Secure API with Authorization Header
- Centralized Error Handling (AppError + asyncHandler)
- MVC Architecture (Models, Controllers, Routes)

---

## 🧠 Concepts Covered

- Authentication vs Authorization
- JWT Structure (Header.Payload.Signature)
- Access vs Refresh Tokens
- Token Expiry & Regeneration
- Middleware-based Route Protection
- Error Handling in Express
- Secure Password Storage

---

## 🛠️ Tech Stack

- Node.js
- Express.js
- MongoDB + Mongoose
- JWT (jsonwebtoken)
- bcrypt
